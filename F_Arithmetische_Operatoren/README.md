# C-Tutorial: Arithmetische Operatoren

Dieses Programm enthält Beispiele für die Verwendung von Arithmetischen Operatoren.

## Beschreibung

- '+' Addition
- '-' Subtraktion
- '*' Multiplikation
- '/' Division
- '%' Modulo
- '++' inkrementieren
- '--' dekrementieren
